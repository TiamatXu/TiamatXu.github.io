## 开发环境设置

启动本地开发服务器：

```bash
pnpm install
pnpm dev
```

## 可用脚本

下面列出开发过程中可用的脚本：

```bash
# Boot local dev server.
$ pnpm dev

# Build docs, then serve locally. This is for testing
# production build in the local environment.
$ pnpm serve

# Run lint via Prettier.
$ pnpm lint

# Run type check via tsc.
$ pnpm type
```
